A simple dream journal to record dreams. The WordCloud module permits the user to generate a word cloud with the content of all entries. It's still a work in progress.

The main function is writediary( ) which parses and saves the content the user wrote on the app to the txt file my_diary.txt. A listbox is then present in the second page to allow for scrolling between entries, although the formatting is still not ideal. The last functionality is the word cloud function defined in the cloud( ) function.
